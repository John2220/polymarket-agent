# Безопасность

- **Никогда** не коммитьте `.env`, приватные ключи Polymarket, реальные API-ключи и файлы с личными ставками (`polymarket_рекомендации.xlsx`, `*.db`, `data/market_cache/*`).
- В репозитории только `.env.example` с пустыми значениями.
- Если ключ когда-либо попал в git, чат или скриншот — **смените его** в кабинете провайдера (The Odds API, Polymarket и т.д.).

Сообщать о уязвимостях можно через Issues (без публикации секретов в тексте issue).
